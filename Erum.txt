# Erum 프로젝트 최종 정리

## 1. 프로젝트 개요

"Erum"은 사용자들이 자신의 경험과 성찰을 기록하고 공유하는 웹 애플리케이션입니다.
초기 기획에 따라 닉네임만으로 로그인하여, 정해진 질문 양식에 맞춰 자신의 생각을 정리하고 다른 사람들과 댓글로 소통할 수 있습니다.
모든 데이터는 서버에 파일 형태로 영구 저장됩니다.

---

## 2. 주요 기능

- **사용자 인증**
  - 비밀번호 없이 닉네임만으로 로그인합니다.
  - "로그인 정보 저장" 옵션을 통해 로그인 상태를 유지할 수 있습니다.(체크 시 브라우저를 닫아도 유지, 미체크 시 브라우저를 닫으면 로그아웃)
  - 메인 페이지에서 로그아웃할 수 있습니다.

- **게시물 작성 및 조회**
  - 아래 항목들로 구성된 게시물을 작성할 수 있습니다.
    - **제목**: 필수 항목
    - **내용**: 선택 항목 (입력하지 않으면 상세 페이지에 표시되지 않음)
    - **4가지 필수 질문**:
      1. 무엇을 경험하였는가?
      2. 이번 경험을 통해 내가 새롭게 알게 된 점은 무엇이었는가?
      3. 내가 한 선택이나 행동중 다시 돌아간다면 바꾸고 싶은 것은?
      4. 이 경험이 나의 꿈과 어떤 연결고리를 만들어 줬는가?
  - 메인 페이지에서 모든 게시물의 제목 목록을 볼 수 있습니다.
  - 상세 페이지에서 게시물의 모든 내용과 댓글을 확인할 수 있습니다.
  - 텍스트 입력 시 줄바꿈(엔터)이 화면에 그대로 유지됩니다.

- **댓글 기능**
  - 각 게시물 상세 페이지에서 댓글을 작성하고 조회할 수 있습니다.

- **관리자 기능**
  - `root` 닉네임으로 로그인 시 관리자 권한을 가집니다.
  - 관리자는 모든 게시물과 댓글을 삭제할 수 있습니다.

---

## 3. 기술 스택 및 구조

- **프론트엔드**: React, Bootstrap, Axios, React Router
- **백엔드**: Node.js, Express
- **데이터 저장**: 서버의 `db.json` 파일을 통해 데이터 영구 저장
- **프로젝트 구조**: `client` (프론트엔드)와 `server` (백엔드) 폴더로 분리되어 있습니다.

---

## 4. 실행 방법

1.  프로젝트 최상위 디렉토리(`Erum_이룸`)에서 터미널을 엽니다.
2.  `npm install` 명령어를 실행하여 필요한 모든 패키지를 설치합니다.
3.  `npm start` 명령어를 실행하면 프론트엔드와 백엔드 서버가 동시에 시작됩니다.
4.  웹 브라우저에서 `http://localhost:3000` 주소로 접속합니다.

---

## 5. 초기 기획 내용 (원본)

사이트에 접속하면 로그인창이 뜨고, 비밀번호 없이 닉네임만으로 로그인이 가능하다.
로그인후에는 중간에 있는+버튼으로 일기를 작성할수있고 일기 작성 창에는 제목입력창(1줄)과 내용 입력창이 있다.
모두 입력후 등록 버튼을 누른다면 저장되어 로그인때 입력한 닉네임으로 업로드된다.
업로드 된 개시물은 사이트에 접속한 모두가 볼 수 있다.
개시물에는 뎃글을 달수있으며 뎃글은 제목없이 내용과 입력한 사용자의 이름으로 이루어진다.
개시물을 원한다면 더보기를 눌러 모든 내용과, 댓글을 큰 화면으로 볼수있다.
이 사이트는 우리의 꿈을 이루기 위한 과정을 기록하는 창고로 작동한다.
